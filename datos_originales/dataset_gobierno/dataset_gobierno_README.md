# Dataset original: Proceso Constituyente Abierto a la Ciudadanía

## Metadatos

| Campo | Valor |
|-------|-------|
| Título | Proceso Constituyente Abierto a la Ciudadanía |
| Fuente | Ministerio Secretaría General de la Presidencia |
| Creación | 19/08/2016 |
| Última actualización | 10/01/2017 |
| URL oficial | https://datos.gob.cl/dataset/proceso-constituyente-abierto-a-la-ciudadania |
| Licencia original | Datos abiertos del gobierno de Chile |

**Descripción:** Datos y resultados del Proceso Constituyente Abierto a la Ciudadanía en encuentros locales y cabildos provinciales y regionales. Los datos fueron anonimizados a efectos de resguardar la identidad de los participantes.

> **Nota:** Este repositorio incluye únicamente los datos de las instancias de cabildo (ELA, provincial y regional). La instancia de Consulta Individual no se incluye porque presenta un sesgo estructural: al ser una encuesta en línea sin debate, el comportamiento de las respuestas "otros" difiere significativamente de las instancias de cabildo, donde el debate grupal fue parte esencial de la metodología.

**Palabras clave:** Cabildo, Constitución, Encuentro local
